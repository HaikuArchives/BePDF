Installation

Installing and Running the BePDF Program

To install BePDF, simply double click the zip archive, choose a destination in the Expand-o-matic dialog, and choose "Expand." Then open the expanded package and follow the install instructions.

To run BePDF, locate the directory where BePDF was installed and double click the "BePDF" icon. Also, BePDF may be started by double clicking any PDF file.

Installing the BePDF Deskbar Replicant

A Replicant can be added to the deskbar; this allows one to open BePDF from an icon in the deskbar. The shell scripts "Add BePDF to Deskbar" and "Remove BePDF from deskbar" located in the BePDF directory can be used to add/remove the BePDF replicant to/from deskbar, respectively.
